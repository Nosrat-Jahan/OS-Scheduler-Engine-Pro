# ⚙️ OS Scheduler Engine Pro (v6.6.6)

**OS Scheduler Engine Pro** is a high-performance CPU scheduling simulator engineered to visualize core Operating System concepts. Built as a cornerstone for a **BSc in CSE** portfolio, this platform provides an interactive way to understand how processes are allocated CPU resources using industry-standard algorithms.

---

## 🚀 Key Features
- **Dual Algorithm Core**: Simulate both **First-Come, First-Served (FCFS)** and **Round Robin (RR)** scheduling logic.
- **Dynamic Gantt Chart**: Real-time visual representation of process execution flow with precise time-stamping.
- **Smart UX Integration**: 
  - Supports **Enter Key** triggers for faster simulation.
  - **Instant Reset** functionality to clear all metrics without page reloads.
  - **Auto-Clear Inputs**: Smart quantum box that clears on focus for seamless data entry.
- **Performance Analytics**: Automated calculation of **Waiting Time (WT)** and **Turnaround Time (TAT)**.
- **Responsive Theme**: Switch between **Dark Mode** and **Light-Gray Mode** for optimal visibility.

## 🛠️ Tech Stack
- **Backend:** Python 3.x / Flask Framework
- **Frontend:** HTML5, Modern CSS3 (Variables & Flexbox), JavaScript (ES6+)
- **Logic:** Asynchronous execution and event-driven UI updates.
