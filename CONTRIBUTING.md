# Contributing to OS Scheduler Engine Pro

Thank you for showing interest in improving this CPU Scheduling Simulator! Contributions from the developer community are what make open-source projects thrive.

## How to Get Involved

### Bug Reports & Feedback
- Check the **Issues** tab to see if the bug has already been reported.
- If not, open a new issue with a clear title and a detailed description of the bug.

### Feature Requests
- I am open to adding more algorithms (like SJF or Priority Scheduling). Please open an issue with the tag `enhancement` to discuss your ideas.

### Pull Request (PR) Process
1. **Fork** the repository to your account.
2. Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
3. Maintain the **v6.6.6** coding style and documentation standards.
4. **Commit** your changes with clear messages.
5. **Push** to the branch and open a **Pull Request**.

## Academic Integrity
As this project is part of a professional **BSc in CSE** portfolio, please ensure all contributions are original and properly documented.
